# LogParser Application

Build the application using command :
	 mvn clean install [ This will build the application and also run the JUnit test cases ]

# Running the Application

java Launcher  or In IDE, right click on Launcher class and select Run As Java Application

Input file is located at : src/main/resources/programming-task-example-data.log

# Tools

	Java 1.8
	Maven
	Jacoco Plugin ( JUnit Code Coverage : /target/site/jacaco/index.html )
	JUnit
	Eclipse/IntelliJ
